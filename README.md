CollabSphere is a desktop-based application that helps students share their project ideas and collaborate with peers. The platform streamlines project discovery, team formation, and communication by allowing students to log in with their IDs, post their ideas, and join projects that interest them.

Features:
-> Student Login System
    Secure login using student IDs to access the platform.
-> Post Project Ideas
    Students can submit their project ideas with descriptions.
-> View Other Projects
    All posted ideas are visible to other logged-in students.
-> Join Request System
    Students can send a request to join a project.
    Project owners can approve or deny collaboration requests.
-> Simple & Intuitive UI
    Built using Java Swing for a clean and user-friendly interface.

Tech Stack:
-> Java (Core & OOP Concepts)
-> Swing (GUI)
-> MySQL (Database)
-> JDBC (Database Connectivity)
